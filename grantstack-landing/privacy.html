Privacy notice

GrantStack stores submitted project inputs to generate and display an incentive screening report. Do not submit confidential trade secrets, regulated personal data, or information you are not authorized to share. Pilot submissions may be reviewed to improve report quality.

GrantStack also records basic first-party usage events such as page views, CTA clicks, sample-report views, submission outcomes, and report actions. These events are used to understand activation and product quality; they are not sold to third parties.

Project report links use private access tokens. Treat each report link as sensitive. A production paid workspace should add user accounts, deletion controls, and a formal data processing agreement.
