The painful workflow

Incentive diligence starts while the project is still incomplete.

Expansion teams need a first-pass answer before every detail is final. The usual path is slow: gather scattered facts, search program pages, ask advisors basic fit questions, and hope no obvious incentive category was missed.

01

Program fit is fragmented

Eligibility depends on location, jobs, wages, capex, timing, industry, and local discretion.

02

Generic AI is not a workflow

Prompts do not provide private report links, status tracking, retries, source refresh, or an audit trail.

03

Manual review is expensive too early

Specialists are still needed, but not every rough expansion idea deserves a full advisory sprint.

What the buyer receives

A source-backed memo, not a vague program list.

The report is designed for the first buyer conversation: what looks promising, what could break eligibility, what facts are missing, and which official sources need validation.

  • Cited program recommendations with source URLs.
  • Strengths tied to jobs, capex, facility type, and location.
  • Risk flags before teams overstate incentive value.
  • Next diligence questions for finance, site selection, or advisors.

How it runs

Built like an async product, not a one-off AI form.

01

Submit specs

Capture location, capex, jobs, facility type, wage, timeline, and competing sites.

02

Queue work

API Gateway and Lambda return `202 Accepted`, then SQS handles processing asynchronously.

03

Screen evidence

The worker checks the active source catalog and is provider-ready for LLM/vector retrieval.

04

Render report

DynamoDB stores the status and report behind a private tokenized result link.

Why this exists

The obvious alternatives each miss part of the job.

Spreadsheets

Good for tracking, weak for turning raw project specs into a cited decision-support memo.

ChatGPT alone

Useful for brainstorming, but not tied to private reports, retries, source refresh, or status flags.

Consulting from day zero

Important for final validation, but often too heavy before a project has cleared basic fit.
